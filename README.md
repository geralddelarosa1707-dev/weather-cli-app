# Weather CLI App
Python CLI app that provides real-time weather, forecast, and air quality information using the OpenWeather API.

## Features
- Current weather search
- Forecast search
- Air quality checker
- Search history system
- Celsius/Fahrenheit support
- Logging and error handling

## Tech Used
- Python
- OpenWeather API
- JSON
- Requests
- Python-dotenv
- Logging
- Object-Oriented Programming (OOP)

## How to Run
```bash
create a .env file:
  OPENWEATHER_API_KEY=your_api_key_here
  
Run the app:
  python app.py
```

## Notes
- Data is stored locally using JSON files
- Built for learning APIs, OOP concepts, and backend concepts
